PARA ATUALIZAR O GITHUB PAGES

Substitua ou envie TODOS os arquivos deste pacote para a raiz do repositório.
Não envie somente o index.html, pois agora o projeto usa app.js, styles.css,
service worker, manifesto e ícones separados.

ARQUIVOS ESSENCIAIS
- .nojekyll
- index.html
- styles.css
- app.js
- presentation.html
- config.js
- manifest.webmanifest
- sw.js
- icon.svg
- icon-192.png
- icon-512.png

SUPABASE
- Mantenha config.js com a URL e a chave pública do seu projeto.
- Nunca use a chave service_role.
- Execute novamente o schema.sql para criar a tabela de histórico online.
- O schema.sql pode ser executado novamente com segurança.

NOVIDADES
- data da reunião e início real corrigido;
- reset preserva a reunião online;
- finalizar para todos os cronômetros e arquiva de verdade;
- histórico sincronizado junto da reunião online;
- backup completo para baixar e restaurar;
- restauração compatível com JSON das versões anteriores;
- funcionamento offline/PWA;
- opção para ocultar o nome do orador na apresentação;
- código separado em HTML, CSS e JavaScript para facilitar manutenção.

BACKUP
Use “Baixar backup” para salvar reunião atual + histórico.
Use “Restaurar backup” para selecionar o arquivo JSON e confirmar a restauração.
